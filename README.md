# 🎮 FrivGameProyecto - Plataforma de Minijuegos

Una aplicación multiplataforma desarrollada con **.NET MAUI** y **C#**, diseñada para ofrecer una experiencia de juego fluida tanto **online** como **offline**.

> **ESTADO DEL PROYECTO:** Actualmente en desarrollo activo.

## ✨ Características Principales
* **Variedad de Minijuegos:** Incluye juegos clásicos y de lógica como:
    *  **Cazar al topo** (Reflejos)
    *  **Wordle** (Lógica y vocabulario)
    * *...y más en camino!*
* **Gestión de Usuarios:** Registro e inicio de sesión para personalizar la experiencia.
* **Rankings Globales:** Compite con otros jugadores y mira quién tiene la mejor puntuación.
* **Modo Offline:** Juega en cualquier lugar sin necesidad de conexión a internet.

## 🛠️ Tecnologías utilizadas
* **Lenguaje:** C#
* **Framework:** .NET MAUI (Multi-platform App UI)
* **Entorno:** Visual Studio 2022

## 🚀 Cómo ejecutar el proyecto
1. Clona el repositorio o descarga el ZIP.
2. Abre el archivo `.sln` en **Visual Studio 2022**.
3. Asegúrate de tener instalado el workload de **.NET MAUI**.
4. Pulsa **F5** para compilar y ejecutar en el emulador o dispositivo local.
